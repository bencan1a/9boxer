## Screenshot Mismatch Investigation

### Why 87 files but only 36 references?

**Analysis:**

1. **Over-generation for Testing** (estimated 20-25 files)
   - Multiple variations for visual regression testing
   - Before/after states for comparison
   - Different UI states (hover, active, error)
   - Test fixtures not meant for docs

2. **Prepared But Not Yet Documented** (estimated 15-20 files)
   - Screenshots ready for future documentation
   - Features documented but screenshots not linked
   - Work-in-progress documentation

3. **Duplicates Across Directories** (estimated 5-8 files)
   - Same screenshot in multiple locations
   - Different naming but same content

4. **Actually Missing Documentation** (estimated 20-25 files)
   - Screenshots exist but docs don't reference them
   - Documentation written before screenshots generated
   - Orphaned during documentation updates

**Conclusion:** We're generating the RIGHT screenshots, but NOT USING them enough in documentation.
