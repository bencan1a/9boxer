# Icon Files

This directory contains platform-specific icon files for the 9Boxer application.

## Current Files

- `icon.png` (512x512) - Source icon used for all platforms
- `icon.ico` - Windows icon file (multi-resolution)
- `icon_*.png` - Various sizes (16x16 to 1024x1024) for different use cases
- `icon.iconset/` - macOS iconset directory for .icns generation

## Icon Generation

The app logo is generated from the Logo React component (`src/components/branding/Logo.tsx`) using the `gradient-bordered` variant.

### Generate All Icons (Recommended)

To regenerate all icon files from the Logo component:

```bash
cd frontend

# 1. Generate PNG files at all required sizes
npm run generate:icons

# 2. Create Windows .ico file (multi-resolution)
python scripts/create_icon.py

# 3. Create macOS iconset directory
python scripts/create_icns.py

# Note: On macOS with iconutil installed, step 3 will also create icon.icns
# On Windows/Linux, you'll need to convert manually or use electron-builder's auto-conversion
```

### Manual Steps (if needed)

**Windows .ico Creation:**
```bash
cd frontend
python scripts/create_icon.py
```

**macOS .icns Creation:**

The `icon.iconset` directory is already created by the generation script. You have several options to create the final `.icns` file:

**Option 1: Let electron-builder handle it (easiest)**

Electron-builder can automatically generate `.icns` from `icon.png` during the build process. This works on all platforms.

**Option 2: On macOS (using iconutil)**
```bash
cd frontend/build
iconutil -c icns icon.iconset -o icon.icns
```

**Option 3: Online Converter**
1. Zip the `icon.iconset` directory
2. Go to https://cloudconvert.com/
3. Convert iconset.zip to .icns
4. Download and save as `frontend/build/icon.icns`

## Logo Component

The logo design is defined in `src/components/branding/Logo.tsx` with 11 different variants. The `gradient-bordered` variant is used for application icons:

- Gradient background (blue → purple) matching splash screen
- 3x3 grid with clear borders
- Transparent gaps between boxes
- Glowing center box
- Responsive to light/dark mode

To change the logo design, modify the Logo component and regenerate icons.

## Usage in Build

The icons are referenced in `electron-builder.json`:
- Windows: `build/icon.ico` ✅ (auto-generated)
- macOS: `build/icon.icns` ⚠️ (auto-generated by electron-builder from icon.png)
- Linux: `build/icon.png` ✅ (auto-generated)

## Troubleshooting

**Icons not updating in built app:**
1. Clean the build directory: `rm -rf frontend/build/*.png frontend/build/*.ico`
2. Regenerate icons: `npm run generate:icons`
3. Rebuild the application

**macOS .icns issues:**
- If electron-builder fails to auto-generate, create manually using iconutil on macOS
- Or use an online converter (see Option 3 above)
- The icon.iconset directory is already prepared with all required sizes
