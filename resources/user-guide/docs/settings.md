# Application Settings

The Settings dialog allows you to customize your 9Boxer experience with various preferences and configuration options.

## When to Use This

### Common Scenarios

You'll want to use Settings when:

- **Adjusting for your environment** - Switch to dark mode for extended work sessions or light mode when presenting
- **Reducing eye strain** - Enable dark mode for low-light conditions or late-night calibration prep
- **Screen sharing** - Switch to light mode for better visibility during video calls and presentations
- **Following your preferences** - Set Auto mode to automatically match your system theme throughout the day
- **First-time setup** - Configure your preferred theme when you first start using 9Boxer

- [Getting Started](getting-started.md) - Initial setup and configuration when you first use 9Boxer
- [Preparing for Talent Calibration](workflows/talent-calibration.md) - Consider using light mode when sharing your screen during meetings

## Appearance Settings

### Theme Mode

9Boxer offers three theme options to suit your preference and work environment:

!!! tip "Default Setting"
    The application defaults to **Auto (Follow System)** mode, which automatically matches your operating system's theme preference.

### How to Change Theme

1. **Open Settings** by clicking the gear icon (‚öô)
2. **Navigate to Appearance section** (selected by default)
3. **Select your preferred theme mode**:
    - Click the radio button next to your choice
    - The theme updates immediately
4. **Close the dialog** - your preference is saved automatically

### Theme Selection Indicator

The Settings dialog shows your current theme selection with:

- **Current Selection box** - Displays your chosen theme mode with icon
- **Active theme indicator** - When using "Auto" mode, shows which theme (light or dark) is currently active based on your system settings
- **Real-time preview** - The application theme changes immediately as you select different options

### Theme Persistence

Your theme preference is:

- **Saved automatically** when you make a selection
- **Persists across sessions** - reopening the app remembers your choice
- **Stored locally** in your browser's localStorage
- **Independent of other settings** - changing theme doesn't affect your data or filters

### System Theme Detection (Auto Mode)

When using **Auto (Follow System)** mode:

- 9Boxer detects your operating system's theme preference
- **Windows**: Follows Windows theme settings (Settings ‚Üí Personalization ‚Üí Colors)
- **macOS**: Follows macOS appearance (System Preferences ‚Üí General ‚Üí Appearance)
- **Linux**: Follows desktop environment theme settings
- **Dynamic updates**: Theme changes automatically when you change your OS theme (no app restart needed)

The Settings dialog shows which theme is currently active: "Currently using light theme (from system)" or "Currently using dark theme (from system)"

## View Controls

### Zoom Controls

Adjust the grid view size for different scenarios using zoom controls in the top-right corner of the grid area:

**Zoom Buttons:**

- **+ (Zoom In)** - Makes everything larger for better readability
- **‚àí (Zoom Out)** - Fits more content on screen for overview
- **Reset button** - Returns to 100% default zoom
- **Percentage display** - Shows current zoom level (e.g., "125%")

**Keyboard Shortcuts:**

- `Ctrl` + `+` (or `Cmd` + `+` on Mac) - Zoom in
- `Ctrl` + `‚àí` (or `Cmd` + `‚àí` on Mac) - Zoom out
- `Ctrl` + `0` (or `Cmd` + `0` on Mac) - Reset to 100%
- `Ctrl` + scroll wheel - Zoom in/out smoothly

**When to Use Zoom:**

- **75%-90%** - Get a bird's-eye view of many employees at once
- **100%** - Normal working view (default)
- **110%-150%** - Focus on specific employees, presentations in bright rooms
- **Combine with filters** - Zoom in while viewing filtered groups for better focus

**Zoom Persistence:**

Your zoom level is saved automatically and persists between sessions.

!!! tip "Zoom for Different Tasks"
    - **Quick overview** - Zoom out to 75% to see distribution patterns
    - **Normal work** - Keep at 100% for comfortable reading
    - **Presentations** - Zoom to 125-150% for better visibility on projectors

### Full-Screen Mode

Use full-screen mode for distraction-free work and presentations:

**How to Activate:**

- Click the **full-screen button** in the top-right view controls
- Press `F11` on your keyboard
- The application fills your entire screen, hiding taskbar and window chrome

**How to Exit:**

- Press `Esc` key
- Press `F11` again
- Click the full-screen exit button

**When to Use Full-Screen:**

- **Calibration presentations** - Display the grid on projectors in large meeting rooms
- **Screen sharing** - Maximize visible space during video conferences
- **Focused review sessions** - Remove distractions for deep work
- **Large room displays** - Present on big screens with better visibility

**Combine with Zoom:**

Use full-screen mode together with zoom controls for the best presentation experience:

1. Enter full-screen mode (`F11`)
2. Adjust zoom level to 125-150% (`Ctrl` + `+`)
3. Use filters to focus on specific groups
4. Present with maximum visibility

!!! tip "Presentation Setup"
    Before sharing your screen in a calibration meeting:
    
    1. Apply filters for the team you're reviewing
    2. Press `F11` for full-screen
    3. Zoom to 125-150% for better readability
    4. Press `Esc` when done to return to normal view
