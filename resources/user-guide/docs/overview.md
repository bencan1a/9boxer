# Overview

9Boxer is a desktop application that helps you visualize and manage talent using the 9-box performance/potential grid.

---

## Download 9Boxer

## Get Started in 30 Seconds

1. **Download** the app for your platform
2. **Open** 9Boxer
3. **Load** sample data to explore features

That's it. You'll see your first talent grid and can start exploring immediately.

**Ready to dive in?** [Try the 2-minute quickstart](quickstart.md)

## What You Can Do

| Capability | Description |
|------------|-------------|
| **Visualize Talent** | Plot employees on a 3x3 grid by performance and potential |
| **Make Rating Changes** | Drag and drop employees to adjust their positions |
| **Validate Ratings** | Use Donut Mode to challenge center box placements |
| **Analyze Distribution** | Review statistics and spot rating anomalies |
| **Run Calibration Meetings** | Filter by team, track decisions, document rationale |
| **Export Results** | Save changes back to Excel with full audit trail |
